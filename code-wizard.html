Senior Project:
Code Wizard

    Over the course of 20 weeks, I designed, implemented, and tested a 2D puzzle game inspired by “Baba Is You.” This game aims to introduce middle school students to programming logic and thinking, providing an engaging and interactive way for them to learn essential coding skills. Players can manipulate how objects in the game behave to solve puzzles; for example, they can take control of the enemy or enable other objects to be moved, adding a dynamic layer of problem-solving to the experience. You can read the full report here, and you can try it for yourself on itch.io
